# Gulp starter 

Heyoo; just a slick little boilerplate to start projects with. 

# Testing 

To run tests just use the gulp task; pipe into faucet for better results. 
``` 
gulp tests | faucet 
``` 

Or you can just use the npm script. It's a little faster. 
``` 
npm run test | faucet
```
